About Advaita

Advaita is a Plymouth, MI biotechnology start up, founded by Wayne State University Professor Dr. Sorin Draghici. The people at Advaita have developed a software application called Pathway-Guide which allows for researchers to process gigabytes of genomic data in short periods of time.

Licensing Management Module

Our client, Advaita required a means of managing their current licenses for their application (PathWayGuide). Before the Licensing Management Module was built, customers could download the entire application and have access to all of it, regardless of what they purchased. What we where able to do was create a module that would generate encrypted license files which the customer would be able to install on their pc in order to use the application. Several things that had to be considered while developing the Licensing Management Module Included :
• Creating a solution that would work off line.
• Needs to work with a variety of Operating Systems in mind.
• A method of searching previous configurations and history was required.
• Room for expanding the license option by pulling information from Advaita's data.
• A relatively powerful method of encrypting the license was also required.

The Solution

We built the main part of our application called Sales module which would allow for the sales team to take care of creating licences for customers.
Using the Sales Module to create a license for a client generates an encrypted file along with an installer and help file inside of a zip file which can then be sent to the customer.
The application also features a search function for finding previous licenses based on search criteria and viewing them.
Some behind the scenes features include an update function for updating the data set for the application from Advaita's data. Also some classes that Advaita can use to read and process the license when on the clients computer.
