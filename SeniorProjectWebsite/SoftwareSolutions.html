Computer Science: SoftwarE Visisualization and Evolution REsearch GROUP (SEVERE GROUP)

The SEVERE Group is over seen by:
Dr. Andrian Marcus, Assistant Professor
Dr. Václav Rajlich, Professor
Dr. Marwan Abi-Antoun, Assistant Professor

Our project is overseen by Laura Moreno who is a third year Ph.D. student. Our client is one out of many software engineering researchers that are interested in software maintenance and evolution. Specifically in concept location and traceability link recovery.

Their areas of research are software maintenance and software evolution. According to the available data, these two processes consume an overwhelming portion of software engineering resources and therefore a better understanding may have a substantial impact on software quality and productivity. Both software maintenance and evolution are based on repeated changes in the software, where each change typically affects only a small part of software. They are working on developing theoretical models, software tools, environments, and case studies to support these changes.

They are particularly interested in:
• concept location
• software visualization
• incremental change
• change propagation
• documentation
• recovery of traceability links
• application of information retrieval methods to support software analysis
• cognitive theories of program comprehension

Project Description

Our project required building a repository for data about changes and their corresponding issues. The user of this repository will introduce data manually via a web based user interface. They should also be able to run reports about the collected data and simple analysis (such as, finding parts of the code that were changed for multiple issues, or retrieving only the issue title). The system should also check for inconsistencies automatically. Some data will be entered in forms of links to repositories or issues in the issue tracker system, in which case the software should collect automatically the data from that URL.
