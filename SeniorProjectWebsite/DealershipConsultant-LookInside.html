Take an inside Look:
Here you will get a visual of what a user would be able to see in the application.

Login Page

Dealership Selection

After logging in the consultant will have a set of dealerships assigned to him by his manager. The consultant will have the choice to select a dealership to prepare for his meeting.

Dealership Details

The dealership detail page is not only made to organize meetings, but also to assisst in collecting data. Using the dealership detail page the consultant will be able to view specific action plans that the consultant will be speaking to the dealer about.

Action Plan Details

The action plan details page allows the consultant to view the set of recommendations associated with the action plan and a number of details pertaining to all of the recommendations contained inside it. From this page the consultant can record the dealer’s reaction to the recommendations, record the atmosphere during the discussion with the dealer, record is the dealer was on time, review additional notes regarding the action plan, decide upon a specific process to be utilized to generate improvements and meet objectives, and keep track of what the dealer agrees to do to generate improvements. This data will be utilized for further analysis and, therefore, must be logically stored back into in a database for analysis.

Recommendation Details

The recommendation details page allows the consultant to view a number of fine grained details regarding a specific target index, such as Sales numbers. From this page the consultant can view a description of the index, a graph and a table representing the data points collected, additional notes that are specific to this index, and the dealer's response to the plans to improve this index.
