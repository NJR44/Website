Licensing Management Module

Our client, Advaita required a means of managing their current licenses for their application (PathWayGuide). Before the Licensing Management Module was built, customers would download the entire application and have access to all of it, regardless of what they paid for. What we where able to do was create a module that would generate encrypted license files which the customer would be able to install on their pc in order to use the application. Several things that had to be considered while developing the Licensing Management Module Included :
• Creating a solution that would work off line.
• Needs to work with a variety of OS in mind.
• A method of searching previous configurations and history was required.
• A relatively powerful method of encrypting the license was also required.

Technologies:

• The Solution was built in Java Eclipse using Swing for the GUI
• RSA encryption was used for encrypting licenses

About Advaita

Advaita is a Plymouth, MI biotechnology start up , founded by Wayne State University Professor Dr. Sorin Draghici. The people at Advaita have developed a software application called Pathway-Guide which allows for researchers to process gigabytes of genomic data in short periods of time.
